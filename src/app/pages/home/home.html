Angular UFMG Test

Oi, essa SPA escrita em AngularJS v1.5.8 segue os requisitos deste documento. Aqui procuro escrever HTML semântico seguindo as recomendações da especificação W3C. Para tornar mais produtivo o CSS, estou utilizando o preprocessador SASS, no qual sigo o style-guide RSCSS que melhora a componentização do CSS. Atualmente este style CSS está ajudando a resolver vários problemas que eu tinha. Nesta aplicação não utilizei nenhum framework CSS. Por questão de tempo, o layout deve conter falhas, além de não estar 100% responsivo para os mobiles.

  • O JavaScript Angular é escrito em Babel utilizando estes conceitos que auxiliam na manutentabilidade do código
    • Não foi utilizado nenhum modulo, lib ou framework CSS, ressalvando o modulo de rotas, o ui-router.

    Requisitos do projeto

    É necessário ter o NodeJS 4+, NPM 3+ (incluso no NodeJs) e git instalado. Para verificar, use no terminal de sua preferência os comandos:
    node --version && npm --version && git --version

    Instruções de instalação (no terminal)

    Baixe o código fonte:
    git clone https://github.com/henryqrm/angular-ufmg-test/

    Instalando as dependências do projeto:
    npm i

    Talvez você precise do comando sudo

    Executando o projeto:
    gulp serve

    Por fim, acesse via navegador
